Responsive Design — Level 2 · Task 2

Responsive layout with CSS media queries and a mobile‑friendly hamburger menu built with HTML + CSS only.

What’s included

A basic webpage, made responsive, with a CSS-only collapsing navigation.

CSS Media Queries

Layout adapts across breakpoints (desktop, tablet, mobile) — grid becomes a single column on narrow screens.

Hamburger Menu

Navigation collapses into a hamburger icon using the checkbox technique — no JavaScript needed.

Accessible Defaults

Skip link, focus states, and high‑contrast theme. Menu can be toggled via keyboard focus on the label.
